Introduzione a bootstrap

Bootstrap è un framework open source per lo sviluppo di siti e applicazioni web. È stato creato nel 2011 da Mark Otto e Jacob Thornton presso Twitter e successivamente acquistato da GitHub. È uno dei progetti open source più popolari su GitHub, con oltre 110.000 stelle e 47.000 fork.
Esso è ritenuto ancora oggi un valido framework avente una vasta lista di modelli e moduli.
Per garantirne il funzionamento è necessario utilizzare uno dei tanti CDN presenti in rete contenenti le risorse javascript e css ad esso collegate.

Dipendenze:

Head:

Inserire nel tag head del file html le seguenti dipendenze:

<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.2.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-0evHe/X+R7YkIZDRvuzKMRqM+OrBnVFBL6DOitfPri4tjfHxaWutUpFmBp4vmVor" crossorigin="anonymous">

Body:

Inserire alla fine del tag body del file html le seguenti dipendenze:

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.2.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-pprn3073KE6tl6bjs2QrFaJGz5/SUsLqktiwsUTF55Jfv3qYSDhgCecCxMW52nD2" crossorigin="anonymous"></script>
